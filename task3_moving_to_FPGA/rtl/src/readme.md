   # 模块说明：
   - edge_cnn_top: 顶层模块，集成了整个CNN推理流程，负责数据输入输出和模块间的连接。
   - cnn_top_fsm: 控制模块，负责管理CNN推理的各个阶段，包括卷积、池化和全连接层的执行顺序。        
   - feature_buffer_sync: 特征缓冲区，存储卷积和池化的中间结果，支持同步访问。
   - Linear Engine: 全连接引擎，执行全连接层的计算，输出分类结果。
   - weight_rom: 权重存储器，存储卷积和全连接层的权重值。
   - bias_rom: 偏置存储器，存储卷积和全连接层的偏置值。
   - conv_layer_engine_sync: 卷积层引擎，执行卷积层的计算，支持同步访问。
   - pool_layer_2x2_engine_sync: 池化层引擎，执行池化层的计算，支持同步访问。
   - global_avg_pool_layer_engine_sync: 全局平均池化层引擎，执行全局平均池化层的计算，支持同步访问。
   - linear_128x10_sync: 全连接层引擎，执行128输入10输出的全连接层计算，支持同步访问。
   - argmax_10: 取最大值模块，输出最终的分类结果（class_id）。
